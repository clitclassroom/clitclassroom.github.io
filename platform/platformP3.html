SaaS

軟體即服務

Software as a Service

就像「租用軟體」。你不需要安裝，打開瀏覽器就能用。

  • 對象：一般使用者
  • 例子：Gmail, Google Drive, Netflix, 線上遊戲
PaaS

平台即服務

Platform as a Service

提供開發工具和環境，讓工程師寫程式更方便。

  • 對象：程式設計師、軟體開發者
  • 例子：Google App Engine, 翻譯 API, 語音辨識 API
IaaS

基礎設施即服務

Infrastructure as a Service

租用硬體設備（虛擬主機、儲存空間），省去買伺服器和蓋機房的錢。

  • 對象：企業、公司 IT 部門
  • 例子：AWS EC2, Google Compute Engine (租雲端虛擬機)

檔案 III：嵌入式系統平台

Embedded System Platforms

它們是「沉默的臥底」。
外表看起來不像電腦，但內部藏著微型電腦系統。
通常只為了執行單一、特定的任務而存在。

核心機密：MCU

Micro Control Unit (微控制器)

一顆小小的晶片，裡面就整合了 CPU、記憶體和輸入輸出。
像是 Arduino 或 micro:bit 開發板，就是學習嵌入式系統的好工具，掀起了自造者(Maker)潮流。

檔案 IV：科技的雙面刃

Impact & Issues

科技帶來便利，但也帶來了新的問題。
作為未來的數位公民，我們需要具備思辨能力。

3D 列印

正面影響 (Pros)

客製化生產、醫療輔具製作(義肢)、減少庫存浪費。

負面衝擊 (Cons)

列印危險物品(槍枝)、數位盜版、智慧財產權爭議。

穿戴式裝置

正面影響 (Pros)

即時健康監測、VR 虛擬旅遊、個人化運動數據。

負面衝擊 (Cons)

個資外洩隱憂(位置、健康數據)、隨時被監控的壓力。

共享經濟 (Uber/YouBike)

正面影響 (Pros)

資源有效利用、降低購買成本、生活便利。

負面衝擊 (Cons)

衝擊傳統產業(計程車)、缺乏法規管理、勞工權益問題。

數位落差 (Digital Divide)

現象：

不會使用數位設備的人（如長輩、偏鄉居民），在科技社會中變得更弱勢，甚至成為「數位難民」。

例如：不會用手機掛號看醫生、ETC導致收費員失業。
